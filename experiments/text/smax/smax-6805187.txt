どのくらいの速度で通信できる？EMOBILE LTE対応モバイルWi-Fiルーター「Pocket WiFi GL04P」のスピードをチェック【レビュー】
How fast is it? 

世界で初めてのUE Category 4対応のLTE通信機器であるイー・モバイル向けモバイルWi-Fiルーター「Pocket WiFi GL04P」（ファーウェイ製）。今現在、そのフルスペックである下り最大150Mbps／上り最大50Mbps（ともに理論値）を活かせる環境にはありませんが、中のチップが変更されたおかけで、従来環境（UE Category 3 : 下り最大75Mbps／上り最大25Mbps）でも通信パフォーマンスが良くなっていると言います。

そこで、都内住宅地（筆者宅）とJR京浜東北線の駅ホーム3ヶ所の合計4ヶ所における通信速度を測定してみました。EMOBILE LTEとGL04Pの組み合わせではどの程度速度が出るのでしょうか？さっそく紹介していきたいと思います。

■測定方法
測定は、手持ちのThinkPad X200（7455-A56）と「ARROWS X ARROWS X F-10D」で行いました。前者では、Radish Network Speed Testingの東京サーバーを利用しました。X200とGL04Pの接続は、USB、無線LAN（Wi-Fi）それぞれ試しています。後者では、Speedtest.netアプリで沖縄・那覇のサーバーとつないだものと、RBB TODAY SPEED TESTアプリでそれぞれ測定しています。

■測定結果
 測定結果は以下の通りです。単位はMbpsです。なお、ARROWS X F-10DでのSpeedtest.netにおける測定は、すべてkbpsで出してしまったので、測定結果を1024で割った値を四捨五入して入力しています。


こうしてみると、X200において、USB接続とWi-Fi接続で下りの通信速度差が有意にある（平均値を見れば一目瞭然）のが見て取れます。ARROWS X F-10DでのWi-Fi接続は、そこそこのパフォーマンスを出しているのですが、この差は一体何なのか……？（詳しい方、考えられる理由をコメントして頂けると幸いです。）
 
数値はともかく、ここに掲載していない部分も含めて実感したところでは、LTEならではの低遅延性・高速性はバッチリ感じられます。

テスト中、データが突然やりとりできなくなる、ということもなく、快適そのものでした。普通、駅構内、というと、人がたくさん居て速度が低下傾向にあるのですが、EMOBILEの気合いか、HuaweiのGL04Pのチューニングの妙か（あるいは、その両方か）、駅構内の方が寧ろスムースでした。
 
一部のイー・モバイルショップでは、EMOBILE LTEレンタルサービスを実施しています。もしも、EMOBILE LTEの通信品質が気になる、という方は是非借りて試してみて下さい！
