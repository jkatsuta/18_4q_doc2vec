ちょっとお高い？GALAXY Note SC-05D用アクセサリー「Bumper Orange for SC-05D」を使用してみた【レビュー】[Galaxy_Report]
本当は裸が一番なんだけど……。 

5インチの大画面ディスプレイを搭載し手のひらに収まるギリギリのサイズ感のNTTドコモ向けAndroidスマートフォン「GALAXY Note SC-05D」（サムスン製）。

片手でも文字入力ができ、大画面ディスプレイで「見」るだけでのスマートフォンではなく、「使う」スマートフォンという印象です。この文章もメモアプリ「Evernote」にGALAXY Note SC-05Dで書いています。

今回はそのGALAXY Note SC-05D用の外側を守バンパーを購入しましたので、どのようなものなのか紹介したいと思います。

そもそも、GALAXY Note SC-05Dにはフリップカバーが着いてくるのに何故バンパーが必要なのかという点から。

あくまで個人的な感覚なのですが、フリップカバーをつけると片手持ちがし辛いんですよね。極端に厚みが増す訳ではないのですが、折り返した分の幅の広がりがその原因。

というわけで、フリップカバーなしで使用していたんですが、ふとしたことで机からGALAXY Note SC-05Dを落下。

プラスチック製の箱の角に側面が当たったのか、少しへこみができてしまいました。まあ、そのへこみでへこんだりはしないのですが、背面がツルツルしてるので、いつか使っていて落とすのではと思い、ケースを買うことしました。

ケースと言ってもボディーカラーの白が気に入って購入したので、全体を覆うケースではなくバンパー的なものを購入しようと。

ここで、時間軸を現在に戻して。そして購入したのがANYMODE製「Bumper Orange for SC-05D」です。流通しているカラーバリエーションは、ホワイト、レッド、オレンジの3色。


パッケージの中には、バンパーの他に前面、背面に貼るデコレーションシールも付いてきます。



装着方法は、内側の黒いシリコン製のパーツを先に装着し、ポリカーボネイト製のパーツをその上から掛けます。



前面は黒いシリコンパーツで保護。



背面は、オレンジが栄えるカラーフルなイメージ、デザインが馴染んでいない印象。そしてゴム部分の厚みが背面の最圧部とあまり変わらないのがちょっと微妙。



外観は操作ボタンが別パーツとなっているため、単なるケースとは違う高級感があります。



2つのパーツで構成されているためかなり肉厚となっていることがわかります。



イヤフォン端子周りは広めに空間があるためイヤフォンの接続は問題ありません。



さて、がっちりプロテクションされたGALAXY Note SC-05Dですが、片手持ちするには厚さと幅のサイズアップが限界を超えてます。これではフリップカバーをしない理由が……。

というわけで、Bumper Orange for SC-05Dのメリットとしては裸状態より向上するであろう耐衝撃性と、普段使いにおける傷防止といったところでしょうか。

前面はディスプレイ保護シートを貼り、背面は傷が付いてもドコモショップでリアカバーを購入できるので、比較的綺麗な状態で使い続けることができると思います。
