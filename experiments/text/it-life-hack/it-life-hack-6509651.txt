より強く！より最強に！ 無敵Galaxy Noteをさらに強くする【イケショップのレア物】
「Galaxy Note」は現在、数あるスマホの中でも“最強の端末”の1つと言える製品だ。通常のスマホより、少し大柄ではあるものの、シャツの胸ポケットにも収まる5.3インチのサイズと800×1280ドットの画面は、並み居るスマホと比較して圧倒的な見やすさを誇る。また、おおぶりな本体サイズのおかげで、他機種より容量が大きい2500mAhバッテリーを内蔵しているためバッテリーの持ち時間が長いというメリットもある。

そして動作パフォーマンスが高いのことも大きな特長だ。CPUはデュアルコアで1.5GHz駆動のSnapdragon S3 APQ8060を搭載している。そのおかげでサクサクというよりヌルヌル動く快適さは一度体験してしまうと他の機種に戻れなくなるほどだ。海外ではすでに500万台を売ったGalaxy Noteは男性だけでなく女性にも受け入れられている。

このように最強の呼び名が似合うAndroid端末Galaxy Noteを、さらに快適かつ強化してくれるアイテム3製品紹介しよう。

 
「3200mAh PowerBank Case for Galaxy Note（CASE11024）」 


■バッテリー内蔵の専用ケース「3200mAh PowerBank Case for Galaxy Note（CASE11024）」
最初に紹介するのは「3200mAh PowerBank Case for Galaxy Note」というGalaxy Note専用ケースだ。このケース本体内に3200mAhのバッテリーを内蔵している。Galaxy Note の標準バッテリーの容量が2500mAhなのでフル充電してもまだ余裕があることになる。このケースがあれば単純計算しても内蔵バッテリーの持続時間の倍以上持つことになる。ケース本体にはGalaxy Noteに充電するかどうかを設定するON/OFF スイッチとケースの内蔵バッテリーに充電するための30ピンコネクタポートが用意されている。


 
内蔵バッテリーに充電するための30ピンコネクタポート、左側は充電状態を切り替えるスイッチ。 



 
このようにケースに装着したままカメラも利用できる。 


■ケースに装着したまま各種操作やカメラが利用可能
このケースはGalaxy Noteの内蔵カメラ部分、具体的にはカメラのレンズ周りがくり抜かれているためケースに装着したままでもカメラが使えるようになっている。また、他のスイッチ部分も操作できるように凹みなどが設けられているのでケースを装着したまま利用可能だ。

※ Galaxy Note Sペンを取り出す際には、バッテリー内蔵ケースを外す必要があります。

■製品名：3200mAh PowerBank Case for Galaxy Note（CASE11024）
■販売価格（税込）：5，980円

■主な仕様
サイズ（約）:159（奥行き）×86（幅）×18（高さ）mm
重量（約）:120g
バッテリー容量:3200mAh
入力:5V/500mA
出力:5V/750mA
付属品:30ピン充電用USBケーブル


 
USB Cradle for Galaxy Note （セカンドバッテリー充電対応）（STND10461） 


■セカンドバッテリーまで充電できるUSB充電専用クレードル
Galaxy Note 用のUSB充電用クレードルが「USB Cradle for Galaxy Note （セカンドバッテリー充電対応）」だ。PC経由でUSBポートからの充電が可能なのに加えてUSB-ACアダプタを使えば電源コンセントからも充電できるようになっている。

また予備用のセカンドバッテリーも充電できるようになっており、背面にバッテリーパックをセットしてmicroUSBケーブルでPCやUSB-ACアダプタを使い充電が可能だ。


 
セカンドバッテリー用の充電機能も持っておりmicroUSBケーブルを使って充電できる。 


■製品名：USB Cradle for Galaxy Note （セカンドバッテリー充電対応）（STND10461）
■販売価格(税込)：2，980円
■主な仕様 
サイズ:110（幅）×95（奥行き）×25（高さ）mm
重量:125g
付属品:microUSBケーブル（長さ110mm）


 
MHL HDTV Adapter for Galaxy Note（CABL11017）。 


■Galaxy Noteを画面とサウンドをそのまま大画面テレビに出力するケーブル
Galaxy Note の画面と音声を HDMI 経由でプラズマや液晶の大画面テレビに出力できるケーブルが「MHL HDTV Adapter for Galaxy Note」だ。Galaxy Note で撮影したHD画質の映像や写真をハイクオリティで大画面テレビを使って楽しむことができる。Galaxy Noteのホーム画面やWebブラウザなどのアプリケーション画面も出力できるのでプレゼンテーションなどにも利用できる。使用するには別途HDMIケーブルとGALAXY Noteに付属のmicroUSBケーブルが必要だ。Galaxy Noteに接続したら、電源供給用としてmicroUSBケーブルをPCやUSB-ACアダプタと接続しHDMIケーブルを使って大画面テレビやディスプレイに接続するだけだ。


 
Galaxy Noteに接続して電源用のmicroUSBを接続、あとはテレビやディスプレイと接続するだけだ。 


■製品名：MHL HDTV Adapter for Galaxy Note（CABL11017）
■販売価格(税込)：1，980円

■主な仕様
ケーブル長（約）:18cm（コネクタ部含む）
重量（約）:17g
規格:MHL規格対応
インターフェイス:HDMIコネクタ（メス端子）、microUSBコネクタ（オス端子）、電源供給用microUSBコネクタ（メス端子）
